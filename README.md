# Object Detection and Analytics System

This project detects and analyzes objects in **images and videos** using the YOLOv8 deep learning model.  
The system identifies multiple objects and highlights them with bounding boxes and confidence scores.

The project demonstrates practical applications of **computer vision and deep learning** for real-world object detection tasks.

---

## Features

- Object detection on images
- Object detection on videos
- Bounding box visualization for detected objects
- Adjustable confidence threshold
- Works with multiple object categories (COCO dataset)
- Easy execution using Google Colab
- Object counting (people, vehicles, etc.)

---

## Technologies Used

- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Google Colab
- Deep Learning / Computer Vision
